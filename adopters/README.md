# AAP Adopters

Applications that implement the Agent Address Protocol (AAP).

## How to add your application

1. Create a new file `adopters/your-app-name.md` (use lowercase, hyphens for spaces).
2. Follow the template below.
3. Open a Pull Request.

## Template

```markdown
# Your App Name

## Overview
Brief description of your application and its purpose.

## AAP Implementation

| Item | Details |
|------|---------|
| **Website** | https://your-app.com |
| **Provider** | your-app.com |
| **AAP Version** | 0.02 |
| **Role** | Provider / Consumer / Both |

## Capabilities
- Resolve: yes/no
- Receive: yes/no
- Inbox: yes/no

## Use Cases
How your app uses AAP (e.g., Agent registration, private messaging, feed posts).

## Links
- Documentation: ...
- skill.md (if applicable): ...
```
